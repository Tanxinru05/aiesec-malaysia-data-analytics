# AIESEC in Malaysia — Data & Analytics Portfolio

Data infrastructure and analytics tooling built for **AIESEC in Malaysia's** national Organization Development function — automating performance reporting, visualizing results across **11 Local Committees (LCs)**, and upskilling the organization in data literacy.

> **Role:** Entity Performance Data Analyst & Data Specialist — Organization Development *(Sep 2025 – Present)*
> Dashboards and reports built here serve **50+ stakeholders across 11 entities**.

---

## The problem

AIESEC in Malaysia tracks entity performance across multiple programme and functional pillars every quarter. Before this work, reporting relied on manual spreadsheet consolidation — slow, error-prone, and impossible to scale across 11 entities. The three projects below replace that with an automated, self-serve, insight-driven workflow.

---

## Projects

### 1. 📊 [LC Report Generator](https://github.com/Tanxinru05/lc-report-generator) — 🔗 [Live demo](https://tanxinru05.github.io/lc-report-generator/)

A web app that takes each Local Committee's quarterly data and generates a **vertical dashboard report** automatically — including auto-written entity-development *"Fruit for Thought"* discussion questions, turning raw numbers into leadership-ready commentary.

- Standardizes quarterly reporting for every LC — no manual consolidation
- Auto-generated narrative insights alongside charts
- Privacy-first: all data stays in the user's browser (localStorage); nothing is uploaded
- **Tech:** HTML/CSS/JavaScript · client-side data processing · GitHub Pages

### 2. 📈 [Visualization Project](https://github.com/Tanxinru05/Visualization-Project) — dashboard & tracker portfolio

A portfolio of dashboard and tracker templates for organizational performance work, built across **Looker Studio, Power BI, Tableau, and Google Sheets**. Each project lives in its own branch:

| Project | Branch |
|---|---|
| Marketing Analytics Dashboard | [`Marketing-Analytics-Dashboard`](https://github.com/Tanxinru05/Visualization-Project/tree/Marketing-Analytics-Dashboard) |
| Human Resources (HR) Control Tool | [`Human-Resources-Tracker`](https://github.com/Tanxinru05/Visualization-Project/tree/Human-Resources-Tracker) |
| Organizational Development Tracker | [`Organizational-Development-Tracker`](https://github.com/Tanxinru05/Visualization-Project/tree/Organizational-Development-Tracker) |
| Lifelong Tracker (personal OS, AI-assisted) | [`Lifelong-Tracker`](https://github.com/Tanxinru05/Visualization-Project/tree/Lifelong-Tracker) |

- Covers KPI tracking, reporting automation, and data storytelling
- **Tech:** Looker Studio · Power BI · Tableau · Google Sheets · HTML/JS

### 3. 🎓 [Data Academy Portal](https://github.com/Tanxinru05/data-academy-portal) — internal LMS

A production-ready internal **Learning Management System** built to scale data literacy across the organization, so analytics capability doesn't depend on one specialist.

- Structured learning pathways with **sequential module unlocking** and automatic progress tracking
- QR-verified certificates, gated team-based hackathon, personal dashboards with streaks and badges
- Full **no-code Admin CMS**: content CRUD, analytics by department/LC, CSV export
- **Tech:** Next.js 15 · React 19 · TypeScript · Tailwind CSS v4 · Prisma · PostgreSQL · Auth.js · Recharts
